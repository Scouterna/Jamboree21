API documentation

| Key | Allowed values | Explanation               |
|-----|----------------|---------------------------|
| l   | ints, 0-200    | Limit of posts            |
| d   | floats, 0-30   | Days (or hours)           |
| m   | 0 or 1         | hide minor changes        |
| b   | 0 or 1         | hide changes made by bots |
